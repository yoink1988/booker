Механика:

В выходные дни комнаты не бронируются.
Рабочее время 8-20.

При бронировании повторяющегося события, если время в первый день занято
- не создается вся цеочка событий.

Состоявшиеся события не редактируются и не удаляются.
Событие считается состоявшимся, если время начала события уже наступило.

Изменения в повторяющихся событиях будут применены начиная с выбранного.

Админ может сменить владельца части повторяющихся событий событий.
Тогда каждый из владельцев управляет только своими из этой цепочки.
Админ может редактировать всю цепочку.
